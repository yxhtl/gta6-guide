Trial Run

Chapter: Prologue · Difficulty: Normal · Reward: $2,000

Mission Objectives

  1. Reach the target building in Downtown
  2. Retrieve the package (stealth or loud approach)
  3. Exfiltrate before backup arrives

Tips

Stealth is optional but easier — use the side entrance. If you go loud bring body armor.

Trivia

This mission teaches basic stealth mechanics and introduces the expanded wanted level system.
