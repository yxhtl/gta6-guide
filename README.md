## Why This Exists

GTA6 launches **November 19, 2026**. Pre-orders open **June 25, 2026**.

When the game drops, millions of players will be searching for guides — and the information will be scattered across Reddit threads, ad-heavy wikis, and clickbait YouTube videos. This project aims to be the opposite: fast, clean, and ad-free.

Most sections will be filled with real data within the first week of launch. The pre-order and money guide sections are already live with predictions based on Rockstar's 20-year release history.
